# Acquiring this repo

    cd ~
    git clone https://github.com/guillaumewatteeux/dotfiles

# Require

## On MacOsX

    brew install stow
    brew install antigen

# Setup

    cd ~/dotfiles
    stow zsh
    stow shells
    stow vim

## Vim
Vim plugins require NeoBundle

    git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
